# Docs

Open and edit Google Docs in Vim with Markdown. Currently support for links, bold, italic, and lists. 

## Installation

- Add your own `credentials.json` file into a `./creds/` folder
- Clone repo
- Create virtual environment
- `pip3 install .`

## Usage

`docs open "DOC NAME"`
